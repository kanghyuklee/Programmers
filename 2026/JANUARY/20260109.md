## 명예의 전당 (1)

### **문제 설명**

"명예의 전당"이라는 TV 프로그램에서는 매일 1명의 가수가 노래를 부르고, 시청자들의 문자 투표수로 가수에게 점수를 부여합니다. 매일 출연한 가수의 점수가 지금까지 출연 가수들의 점수 중 상위 k번째 이내이면 해당 가수의 점수를 명예의 전당이라는 목록에 올려 기념합니다. 즉 프로그램 시작 이후 초기에 k일까지는 모든 출연 가수의 점수가 명예의 전당에 오르게 됩니다. k일 다음부터는 출연 가수의 점수가 기존의 명예의 전당 목록의 k번째 순위의 가수 점수보다 더 높으면, 출연 가수의 점수가 명예의 전당에 오르게 되고 기존의 k번째 순위의 점수는 명예의 전당에서 내려오게 됩니다.

### 정답

```python
def solution(k, score):
    answer = []
    k_list = []
    for s in score:
        k_list.append(s)
        k_list.sort(reverse=True)
        
        if len(k_list) > k:
            k_list = k_list[:k]
        answer.append(k_list[-1])
            
    return answer 
```

### heapq

- 최소 힙(min-heap) 자료구조 라이브러리
- 핵심
    - 부모 노드 ≤ 자식 노드
    - 가장 작은 값이 항상 맨 앞 (인덱스 0)
        
        ```python
        [2, 5, 3, 9, 8]
        ↑
        항상 최소값
        ```
        
    - 정렬된 리스트가 아님
        - [0]만 최소값 보장
        - 나머지는 순서 의미 없음
- 사용법
    - `import heapq`
    - 힙에 값 넣기
        
        ```python
        heapq.heappush(heap, value)
        ```
        
    - 최소값 꺼내기
        
        ```python
        heapq.heappop(heap)
        ```
        
    - 최소값 확인하기
        
        ```python
        heap[0]
        ```
        
    - 리스트를 힙으로 변환
        
        ```python
        heapq.heapify(lst)
        ```
        
- 최대힙 → 부호를 바꿔서 사용
    
    ```python
    heapq.heappush(heap, -value)
    max_value = -heapq.heappop(heap)
    ```
    

### heapq 사용 수정 쿼리

```python
import heapq

def solution(k, score):
    hall = []      # 최소 힙
    answer = []

    for s in score:
        heapq.heappush(hall, s)

        if len(hall) > k:
            heapq.heappop(hall)   # 가장 작은 점수 제거

        answer.append(hall[0])    # 현재 명예의 전당 최하위

    return answer
```
