## 시저 암호

### **문제 설명**

어떤 문장의 각 알파벳을 일정한 거리만큼 밀어서 다른 알파벳으로 바꾸는 암호화 방식을 시저 암호라고 합니다. 예를 들어 "AB"는 1만큼 밀면 "BC"가 되고, 3만큼 밀면 "DE"가 됩니다. "z"는 1만큼 밀면 "a"가 됩니다. 문자열 s와 거리 n을 입력받아 s를 n만큼 민 암호문을 만드는 함수, solution을 완성해 보세요.

- 제한 조건
    - 공백은 아무리 밀어도 공백입니다.
    - s는 알파벳 소문자, 대문자, 공백으로만 이루어져 있습니다.
    - s의 길이는 8000이하입니다.
    - n은 1 이상, 25이하인 자연수입니다.
- 입출력 예
    
    
    | s | n | result |
    | --- | --- | --- |
    | "AB" | 1 | "BC" |
    | "z" | 1 | "a" |
    | "a B z" | 4 | "e F d" |

---

### 정답

```python
import string

def solution(s, n):
    l = string.ascii_lowercase
    u = string.ascii_uppercase
    answer = []
    
    for char in s:                              # 1. char가 s에 포함되는 문자라면
        if char not in l:                       # 2. char가 대문자일 경우
            if char == ' ':                     # 3. char가 공백일 경우 공백을 리스트에 넣기
                answer.append(' ')
            else:                               # 4. char가 공백이 아니라면
                idx = (u.index(char) + n) % 26  # 5. 알파벳 위치 26넘어가면 다시 a부터 세기(%)
                answer.append(u[idx])
        elif char in l:                         # 6. char가 소문자일 경우
            idx = (l.index(char) + n) % 26
            answer.append(l[idx])
    
    answer = ''.join(answer)
    return answer
```

1. string 모듈의 ascii_lowercase (소문자), ascii_uppercase (대문자) 사용
    - 숫자 1~0까지는 digits
2. index
    - 해당 리스트의 인자가 몇 번째 순서인지 출력
